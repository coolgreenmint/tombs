   1. 编程语言
   2. 计算平台
 
二. Java的特点

   1. 面向对象且简单易用，快速敏捷开发
   2. 摒弃了C语言指针等一系列晦涩难懂的概念（易用性）
   3. 静态语言（编写时数据类型确定，编译可知错误）
   4. 跨平台性（java编译成字节码，由jvm编译成机器语言执行由此实现跨平台）
   5. 多线程实现比较容易
   6. 高性能（主要是动态编译器，运行时热点代码编译机器码）
   7. 健壮性（异常处理、垃圾回收、强类型机制等）
   8. 容易开发分布式项目（Internet类库支持）
   
三. JDK与JRE区别

   JDK java开发工具包，包含很多工具及开发类库，比jre大
   JRE java运行环境 
   
四. 数据类型

         整型： byte short int long
       
       byte  : 一字节等于8bits,默认值为 0
       short : 占用两个字节，16位， 默认值为 0
       int   : 占用四字节，32位，默认值 为 0
       long  : 占用八字节，64位，默认值为0L
       
       浮点型： float  double
       
       float : 占用四字节，32位，默认值为0.0f
       double: 占用八字节，64位，默认值0.0d
       
        字符型：char
     
       char : 单一的16位Unicode字符
       
         布尔型： boolean
        
       boolean : true或者false 8位
